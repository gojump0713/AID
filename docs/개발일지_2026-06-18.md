## 1. 작업 개요

빈 저장소(README만 존재)를 클론한 상태에서 시작하여, 권역별 선정대학 비전과 틸론 영업 관점을
정리·시각화하는 보고용 정적 웹사이트를 구축하고 GitHub Pages로 배포 완료하였다.
하루 동안 4개의 기능 커밋을 통해 베이스 환경 구축 → 핵심 기능 → 구조 개편 → 콘텐츠 보강 순으로 진행했다.

## 2. 시간순 작업 내역

### 2.1 저장소 클론 및 구조 분석
- `gojump0713/AID` 클론. 초기 상태는 `README.md` 단일 파일.
- 사용자가 사업 개요 **PDF 1종**과 권역별 **비전정리 엑셀 5종**(수도권·강원·충청·부산울산경남·호남제주) 업로드.
- 누락된 환경 설정 파일 목록을 보고 후, 기술 스택을 **Vite + Vanilla**로 확정.

### 2.2 원본 자료 처리
- PDF는 `pdftotext` 추출 시 한글이 깨져(임베드 폰트/인코딩 이슈) 본문 활용 불가 → **엑셀 데이터로 대체** 결정.
- 로컬 Python은 Microsoft Store 스텁이라 사용 불가, `pdftoppm` 부재로 PDF 렌더링도 불가 확인.
- **Node 기반 `xlsx` 파서**로 엑셀 5종을 손실 없이 파싱하여 `src/data/universities.json`으로 구조화.
  - 결과: **5개 권역 / 19개 사업단**, 각 항목에 구분(단독/연합)·대학명·비전·출처·추진방향·틸론 관점 포함.

### 2.3 베이스 환경 구축 (커밋 `204b7f4`)
- 설정 파일: `package.json`, `vite.config.js`, `.gitignore`
- 디자인 시스템:
  - `tokens.css` — 컬러 토큰(주: 다크블루 `#0a2342`·로열블루 `#2546d6` / 포인트: 다크그린 `#14532d`·다크레드 `#7f1d1d`), 타이포·간격·그림자
  - `base.css` — 리셋 + **흑백 아이콘 시스템**(Material Symbols Outlined, `FILL 0/wght 300`의 가는 감성 톤). 컬러 이모지 미사용 원칙.
- 핵심 기능:
  - `map.js` — **인터랙티브 권역 SVG 지도**(5개 권역 스키매틱 타일, 호버·선택·글로우·펄스 효과, 진입 애니메이션)
  - `main.js` — 권역 탭 + 19개 사업단 카드(비전·추진방향·틸론 관점), 지도↔탭 양방향 동기화, 틸론 제품 키워드 하이라이트

### 2.4 내비게이션·사업소개·파트너·출처 (커밋 `204b7f4` 보강)
- 공통 상단 내비게이션(`nav.js`): **AID 사업소개 · 권역별 선정 현황**
- AID 사업소개 페이지 — 사업 개요·권역 구성·틸론 솔루션 맵(iStation/Dstation/DaaS/CAS)
- 권역별 **영업 파트너사** 표시(`partners.js`): 수도권(솔모정보기술·제이원 아이티), 강원(인투정보), 충청(테라인포메이션), 부울경(우리아이티), 호남·제주(싸이버테크)
- 출처 클릭 시 새 창으로 자료 열기

### 2.5 1차 커밋·배포
- main 푸시 후 프로덕션 빌드 → **GitHub Pages 배포**.
- 토큰에 `workflow` 스코프가 없어 Actions 배포 불가 → **`gh-pages` 브랜치 방식(`build_type: legacy`)으로 전환**하여 배포 성공.

### 2.6 구조 개편 (커밋 `43b2d65`)
- **AID 사업소개를 메인(index)으로 승격**, 권역 보고는 `regions.html`로 분리.
- **파비콘 제작**: 브랜드 그라데이션 + 네트워크/허브 모티프 SVG(`public/favicon.svg`).
- **Hero 캔버스 애니메이션**(`hero.js`): AI·Digital 네트워크 파티클(노드/연결선, 마우스 인터랙션, `prefers-reduced-motion`·탭 비활성 정지 대응).

### 2.7 배포 자료 페이지 (커밋 `8f45f00`)
- 내비에 **배포 자료** 추가, `media.html` 신설.
- 유튜브 영상 반응형 16:9 임베드(youtube-nocookie, lazy) — *AI Native Campus로의 전환 : 대학 교육 인프라의 새로운 표준*.

### 2.8 출처 URL·Hero 보정 (커밋 `893e59d`)
- 18개 대학 출처를 **지정 기사 URL**로 1:1 연결(부천대는 검색 유지). 대학명 부분일치 매핑.
- Hero 제목 줄바꿈("전환" 뒤) + `word-break: keep-all`로 한글 단어 잘림 방지.

## 5. 기술적 의사결정 및 트러블슈팅

| 이슈 | 원인 | 해결 |
|---|---|---|
| PDF 한글 추출 깨짐 | 임베드 폰트/인코딩, `pdftoppm` 부재 | 엑셀 데이터로 대체, `xlsx` 파서로 JSON 변환 |
| Python 사용 불가 | MS Store 스텁(WindowsApps) | Node 생태계로 처리 |
| GitHub Actions 배포 불가 | 토큰 `workflow` 스코프 없음 | `gh-pages` 브랜치 배포(legacy)로 전환 |
| 권역 지도 정밀 경계 부담 | 정확한 행정경계 GeoJSON 부재 | 보고 목적의 스키매틱 타일맵으로 설계 |
